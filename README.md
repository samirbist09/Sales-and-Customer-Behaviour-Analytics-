# 📊 Sales & Customer Behavior Analytics Dashboard

## Overview

This project analyzes sales performance and customer purchasing behavior using the Brazilian E-Commerce (Olist) dataset. The goal was to clean and analyze real-world e-commerce data, generate business insights, and build an interactive dashboard using Streamlit.

The project covers the complete data analytics workflow, including data cleaning, feature engineering, exploratory data analysis (EDA), customer behavior analysis, and dashboard development.

---

##  Features

* Data Cleaning and Preprocessing
* Sales Trend Analysis
* Customer Behavior Analysis
* Revenue and Profit Calculation
* Customer Lifetime Value (CLV)
* Repeat Customer Rate Analysis
* Interactive Streamlit Dashboard
* Business Insights and Recommendations

---

## 📂 Dataset

The project uses the Brazilian E-Commerce (Olist) dataset and combines data from:

* Customers
* Orders
* Order Items
* Payments
* Products

After cleaning and merging:

* Total Records: 115,034
* Total Features: 34

---

##  Key Insights

### Sales Performance

* Highest Revenue Month: **May**
* Lowest Revenue Month: **September**
* Top Product Category: **Beauty & Health**

### Customer Behavior

* Repeat Customer Rate: **15.02%**
* Most customers purchased only once.
* A small group of customers generated a large portion of revenue.

### Payment Analysis

* Most Used Payment Method: **Credit Card**
* Digital payment methods dominated customer transactions.
